# Tkinter Calculator

## Description
This is a simple GUI calculator built using Python Tkinter. It supports basic arithmetic operations.

## Features
- Addition, Subtraction, Multiplication, Division
- Clear button
- Error handling
- User-friendly interface

## How to Run
1. Install Python
2. Run the file:
   python calculator.py

## Requirements
- Python 3.x
- Tkinter (comes with Python)
